About Me

이 섹션은 단순한 자기소개가 아니라, 세하라는 이름이 가진 에너지와 스타일을 보여주기 위한 공간입니다. 아래 카드들에 원하는 소개 문장으로 바꾸면 바로 간지나는 소개 사이트가 됩니다.

IDENTITY

세하라는 이름

새로운 것을 만들고, 복잡한 문제를 구조적으로 정리하고, 보는 사람이 감탄하는 결과물을 만드는 사람.

SKILLS

만드는 방식

HTML, CSS, JavaScript를 바탕으로 애니메이션, 인터랙션, UI 구조를 직접 설계해 분위기 있는 결과물을 만듭니다.

MINDSET

추구하는 것

평범한 화면보다 기억에 남는 화면. 단순한 소개보다 인상 깊은 경험. 보기만 해도 매력이 느껴지는 웹사이트.

Showcase

슬라이드로 핵심 특징을 보여주는 구간입니다. 자동 전환도 되고, 버튼으로 직접 넘길 수도 있습니다. 카드 속 문장을 원하는 포트폴리오 소개로 바꾸시면 됩니다.

Immersive UI

스크롤을 내릴수록 분위기가 깊어지고, 요소가 자연스럽게 살아나는 몰입형 UI. 정적인 자기소개가 아니라 한 편의 오프닝처럼 느껴지는 화면을 목표로 했습니다.

Visual Identity

‘세하’라는 이름이 가진 이미지를 하나의 세계관처럼 표현했습니다. 푸른 광채, 미래적인 입체감, 그리고 부드러운 모션이 브랜드 같은 인상을 만들어냅니다.

Motion & Energy

정적인 레이아웃에 그치지 않고, 시각적인 리듬을 넣었습니다. 살짝 떠 있는 카드, 돌아가는 오브젝트, 자동 슬라이드까지 페이지 전체가 살아 움직입니다.

Journey

성장 과정이나 프로젝트 흐름을 멋지게 보이게 해주는 타임라인 섹션입니다. 여기를 주인님 이력이나 활동 내용으로 바꾸면 자기소개 사이트 완성도가 확 올라갑니다.

관심에서 시작

무언가를 단순히 소비하는 것보다 직접 만드는 데 더 큰 재미를 느끼며, 웹과 코딩에 자연스럽게 빠져들었습니다.

감각적인 화면에 집중

기능만 되는 페이지보다, 열자마자 ‘와’ 하는 분위기를 주는 화면을 만들고 싶어서 애니메이션과 인터랙션에 집중했습니다.

나만의 스타일 구축

입체감, 유리 느낌, 부드러운 움직임, 강한 첫인상. 이런 요소들을 조합해 세하만의 톤을 가진 랜딩 페이지를 만들어가고 있습니다.
